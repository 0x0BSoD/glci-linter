# Generate and lint GitLab-CI yaml with GitLab API

```
Usage:
  glci-linter [flags]
  glci-linter [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  hook        With 'hook' subcommand you can add or delete pre-commit hook in repo
  lint        The 'lint' subcommand will try to find and lint gitlab-ci.yaml file in direcory settetd as first arg.

Flags:
  -h, --help   help for glci-linter

Use "glci-linter [command] --help" for more information about a command.
```
